`prybar`: Create temporary `pkg_resources` entry points at runtime
=================================================================

.. .. toctree::
       :maxdepth: 2
       :caption: Contents:


Use `prybar` to temporarily define `pkg_resources` entry points at runtime. The
created entry points exist for the lifetime of the `dynamic_entrypoint` context
manager.
